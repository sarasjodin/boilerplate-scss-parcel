My Development Process

  1. Reused a previous boilerplate (without Sass)
  2. Created the 7-1 patters for Sass and its main.scss file with @use
  3. Installed Sass npm install sass -D
  4. Created a Bootstrap menu for large screens
  5. Created a Bootstrap Offcanvas responsive menu
  6. Realized that I was alreay developing and had to start commiting to GitHub...
  7. Created the GitHub repository!
